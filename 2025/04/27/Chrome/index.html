谷歌或被迫出售 Chrome 浏览器，科技巨头竞相收购

谷歌或被迫出售 Chrome 浏览器，科技巨头竞相收购

背景概述

2025年4月，美国司法部在反垄断诉讼的补救阶段，正式要求谷歌剥离其旗下的 Chrome 浏览器业务。此举旨在打破谷歌在搜索市场的垄断地位，恢复行业竞争。司法部指出，Chrome 浏览器作为用户访问互联网的主要入口，谷歌通过其控制地位，限制了其他搜索引擎的发展机会 :contentReference[oaicite:0]{index=0}。

潜在收购者

雅虎（Yahoo）

雅虎对收购 Chrome 表现出浓厚兴趣。雅虎搜索总经理 Brian Provost 在法庭上表示，浏览器是搜索流量的关键入口，控制 Chrome 将有助于雅虎提升搜索市场份额。据悉，雅虎已在内部开发浏览器原型，并计划在六到九个月内完成开发。然而，收购 Chrome 将为其提供更快速的扩展路径 :contentReference[oaicite:1]{index=1}。

OpenAI 与 Perplexity

人工智能公司 OpenAI 和 Perplexity 也被报道对收购 Chrome 表达了兴趣。尽管具体细节尚未公开，但这表明 AI 公司正在寻求通过控制浏览器入口来增强其在搜索和信息获取领域的影响力。

谷歌的回应

谷歌对司法部的提议表示强烈反对，称强制出售 Chrome 将对消费者造成伤害，增加设备成本，降低隐私保护，并阻碍人工智能的发展。谷歌计划对法院的最终裁决提出上诉，预计整个法律程序可能持续数年 :contentReference[oaicite:2]{index=2}。

结语

Chrome 浏览器作为全球最受欢迎的浏览器之一，其未来归属将对互联网生态产生深远影响。无论最终归属哪家公司，都将重新定义用户的上网体验和搜索引擎市场的竞争格局。我们将持续关注事态的发展，并为读者带来最新报道。
